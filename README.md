# Бизнес-зеркало — 100 вопросов

Бесплатный русскоязычный сайт-разбор. Человек отвечает на 100 жестких вопросов «Да/Нет» и получает карту по 10 спектрам:

- энергия и восстановление;
- дисциплина и доведение до конца;
- мышление и решения;
- риск и смелость;
- деньги и ценность;
- лидерство и влияние;
- коммуникация и границы;
- эмоциональная устойчивость;
- ценности и смысл;
- предпринимательский потенциал.

Перед вопросами есть вводный экран: как отвечать честно, почему не стоит выбирать “правильный” ответ и что ответы считаются в браузере без регистрации и отправки данных.

Во время прохождения справа двигаются живые бизнес-шкалы: оффер, продажи, деньги, команда, операционка и рынок. После ответа показывается, какая зона подросла или просела.

В финале дополнительно собирается панель из 20 качеств на шкалах: энергия, самообладание, дисциплина, доведение, ясность, честность с собой, смелость, гибкость, самоценность, денежная уверенность, границы, близость, эмпатия, коммуникация, устойчивость, внутренний компас, амбиция, ответственность, лидерство и созидательность.

Для людей, которые двигают бизнес, есть отдельный бизнес-срез: оффер, продажи, деньги, команда, операционка и рынок.

## Как открыть

Откройте файл `index.html` в браузере. Сборка и сервер не нужны.

После публикации сайт доступен по адресу:

https://ferixdi-png.github.io/34/

## Как опубликовать

Проект статический, его можно загрузить на GitHub Pages, Netlify или Vercel.

Если GitHub CLI будет установлен и авторизован, репозиторий можно создать так:

```powershell
gh repo create human-spectrum-quiz --public --source . --remote origin --push
```

## Важно

Опрос не является медицинской диагностикой. Это инструмент саморефлексии и первичной личной навигации.
