# Карта человека — 100 вопросов

Бесплатный русскоязычный сайт-опросник. Человек отвечает на 100 простых вопросов «Да/Нет» и получает карту по 10 спектрам:

- энергия и восстановление;
- дисциплина и доведение до конца;
- мышление и решения;
- риск и смелость;
- деньги и ценность;
- лидерство и влияние;
- коммуникация и границы;
- эмоциональная устойчивость;
- ценности и смысл;
- предпринимательский потенциал.

## Как открыть

Откройте файл `index.html` в браузере. Сборка и сервер не нужны.

После публикации сайт доступен по адресу:

https://ferixdi-png.github.io/34/

## Как опубликовать

Проект статический, его можно загрузить на GitHub Pages, Netlify или Vercel.

Если GitHub CLI будет установлен и авторизован, репозиторий можно создать так:

```powershell
gh repo create human-spectrum-quiz --public --source . --remote origin --push
```

## Важно

Опрос не является медицинской диагностикой. Это инструмент саморефлексии и первичной личной навигации.
